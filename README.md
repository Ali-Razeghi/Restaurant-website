# Restaurant Website – Modern UI Project

A modern, responsive restaurant website designed as a professional front-end portfolio project.
This project focuses on clean design, user experience, and real-world layout structure.

---

## Live Demo
https://golden-spoon-ali.netlify.app/
---

## Project Overview

This project is a fully responsive restaurant website that showcases a digital menu, featured dishes, and a clean user interface.

It is designed to simulate a real restaurant business website and demonstrate front-end development skills.

---

## Features

* Modern and visually appealing UI design
* Hero section with call-to-action
* Food menu with images and descriptions
* Clean layout and typography
* Responsive design (mobile-friendly)
* Smooth scrolling experience
* Contact section
